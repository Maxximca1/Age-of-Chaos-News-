Age of Chaos - Mod News

Age of Chaos is a mod for Age of History. This website will feature news and updates about the mod.

Download the mod: https://www.moddb.com/mods/age-of-chaos

Versions

  • Version 1.3 – This was the first public release, but it is non-functional.
  • Version 1.4 – This is the working version of the mod. It includes formed civilizations from the previous version, along with new ones.
